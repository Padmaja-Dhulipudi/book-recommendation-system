# Book Recommendation System

This project implements a book recommendation system using a Kaggle dataset.
It uses similarity-based filtering to recommend books to users.

## Tech Stack
- Python
- Pandas
- Scikit-learn

## Dataset
Dataset taken from Kaggle (link here)

## How it works
- Data preprocessing and cleaning
- Feature extraction
- Similarity computation (cosine similarity / KNN)
- Top-N recommendation generation
